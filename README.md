# YouTube Mini

Ung dung React nho de tim kiem va xem video tu YouTube Data API v3.
Khi mo app, danh sach mac dinh se lay cac video nhac remix moi nhat bang cach sap xep theo ngay dang va loc trong nhom Music.

## Chay local

```bash
npm install
npm run dev
```

API key duoc doc tu `.env.local`:

```bash
VITE_YOUTUBE_API_KEY=your_youtube_data_api_v3_key
```

Nen gioi han API key trong Google Cloud Console theo HTTP referrer va chi cap quyen cho YouTube Data API v3.
